⭐ Insurance Customer Segmentation using K-Means Clustering

This project was developed during a Data Science internship to analyze insurance customer data and group customers into meaningful segments using machine learning techniques.

The main goal of the project is to understand customer behavior patterns and help businesses make better decisions for targeted marketing and improved customer engagement.

🛠 Tech Stack
Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
Algorithm: K-Means Clustering
Dimensionality Reduction: PCA (Principal Component Analysis)
📊 Project Workflow
Data Preprocessing: Cleaned the dataset, handled missing values, and applied feature scaling for better model performance.
Exploratory Data Analysis (EDA): Analyzed and visualized customer data to understand patterns and distributions.
Customer Segmentation: Applied K-Means Clustering to group customers based on similar characteristics.
Optimal Clusters: Used the Elbow Method to determine the best number of clusters.
Visualization: Used PCA to reduce dimensions and visualize customer segments clearly.
💡 Key Insights

This segmentation helps in understanding different customer groups, which can be used to:

Improve targeted marketing strategies
Enhance customer retention
Optimize insurance product offerings
🎯 Objective

To apply unsupervised learning techniques for meaningful customer segmentation and derive business insights from insurance data.
